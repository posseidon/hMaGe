Practical system of geolocating projectionless maps

This application can be useful in map collections and libraries preserving cartographic material.

Functionalities

Geo-locating Maps

Geolocating projectionless maps that can be implemented without distorting or transforming the original material.

Free-text search

Search for maps by their name and/or extended attributes.

Location-based search

Search for maps based on geolocation objects as polygon.

User management

Simple user management prevent unauthorized actions on geolocated maps.
